# Agile x PMF

自身は2022年頃に大学院でPBL(Project Based Learning) 型教育による新規プロダクト開発の演習をおこなった． 本学は社会人が大半を占め，プロジェクト構成メンバの年齢，職業，職位，経験等が様々であり，属性及び価値観は多様にある．
プロジェクトの始動から巡航に至るまでの活動では，協働作業のためにAgile ScrumとPMF（Product Market Fit）のメソッドを様々活用した．

PBL活動では初対面のメンバーがチームを組成し，意見を集め複数のアイデアを具現化するまでに至ったが．AgileによるチームづくりとPMFによるプロダクト開発のメソドロジーの親和性が高く，産業界でも有効な手法であると実感した．
そこで本サイトでAgile x PMFというテーマで振り返る．
